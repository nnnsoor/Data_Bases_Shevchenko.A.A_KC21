# Farm Management Database — Практична робота №7

## Опис

База даних для управління фермерським господарством.
Реалізована в MS SQL Server 2008.
Назва БД: `Farm_Management_Shevchenko`

## Структура таблиць

| Таблиця | Призначення |
|---|---|
| BRIGADE | Бригади та бригадири |
| CROP | Сільськогосподарські культури |
| FERTILIZER | Добрива |
| LAND_PLOT | Земельні ділянки |
| WORKER | Працівники |
| EQUIPMENT | Сільгосптехніка |
| WORKER_EQUIPMENT_SKILL | Навички керування технікою |
| FERTILIZER_APPLICATION | Записи внесення добрив |

## Файли

| Файл | Вміст |
|---|---|
| `SETUP.SQL` | Створення та налаштування таблиць |
| `INSERT.SQL` | Додавання 110 записів у всі таблиці |
| `UPDATE.SQL` | Оновлення та видалення даних |
| `QUERY.SQL` | 9 запитів: WHERE, AND/OR, LIKE, JOIN, SUBQUERY, GROUP BY |

## Запити (QUERY.SQL)

1. WHERE — ділянки площею понад 100 га
2. AND/OR — ділянки з зрошенням і площею або якісним ґрунтом
3. LIKE — працівники з прізвищем на '-енко'
4. INNER JOIN — ділянки з культурою та бригадиром
5. LEFT JOIN — всі бригади з їх ділянками
6. SUBQUERY — бригади з досвідченими працівниками
7. GROUP BY + HAVING — бригади із загальною площею понад 200 га
8. Складний запит — культура з максимальною врожайністю
9. WHERE + JOIN — добрива на зрошуваних ділянках понад 150 кг
