# Farm Management Database

Навчальний проєкт з дисципліни «Бази даних».  
Предметна область: управління фермерським господарством.  
Реалізовано на базі Microsoft SQL Server з використанням мови T-SQL.

---

## База даних

**СУБД:** Microsoft SQL Server  
**Назва БД:** `Farm_Management_Shevchenko`

### Таблиці

| Таблиця | Опис |
|---------|------|
| `Brigade` | Бригади та їхні бригадири |
| `Crop` | Сільськогосподарські культури та їх характеристики |
| `Land_Plot` | Земельні ділянки з прив'язкою до бригади та культури |
| `Fertilizer` | Добрива та норми їх внесення |
| `Worker` | Працівники з прив'язкою до бригади |
| `Equipment` | Сільськогосподарська техніка |
| `Worker_Equipment_Skill` | Навички керування технікою (зв'язок Worker ↔ Equipment) |
| `Fertilizer_Application` | Записи про внесення добрив на ділянки |

---

## Запити варіанту (QUERY.SQL)

1. Визначити площу під кожну культуру
2. Вивести робітників бригади разом із технікою, якою вони керують
3. Знайти культуру з максимальною врожайністю
4. Знайти бригади з кількістю навичок керування технікою понад середню
5. Скласти список ділянок з невідповідними добривами

## Функції (FUNCTIONS.SQL)

### Scalar Functions
| Функція | Опис |
|---------|------|
| `fn_GetWorkerFullName` | Повертає ПІБ працівника у форматі «Прізвище І.П.» |
| `fn_GetFertilizerDensity` | Розраховує фактичну густоту внесення добрива (кг/га) |
| `fn_GetSoilCategory` | Повертає текстову категорію якості ґрунту ділянки |
| `fn_MaxYieldCropName` | Повертає назву культури з найвищою врожайністю |

### Inline Table-Valued Functions
| Функція | Опис |
|---------|------|
| `fn_GetBrigadeWorkers` | Список працівників вказаної бригади |
| `fn_GetPlotFertilizers` | Добрива, внесені на конкретну ділянку |
| `fn_GetWorkerEquipment` | Техніка, якою може керувати вказаний працівник |
| `fn_AreaByCrop` | Загальна площа та кількість ділянок під кожною культурою |
| `fn_BrigadesAboveAvgEquipmentSkills` | Бригади з кількістю навичок понад середню |

### Multi-Statement Table-Valued Functions
| Функція | Опис |
|---------|------|
| `fn_GetBrigadeStatistics` | Зведена статистика по кожній бригаді |
| `fn_CropAreaDetail` | Деталізація площ під культури по ділянках |
| `fn_FertilizerApplicationReport` | Звіт про внесення добрив зі статусом відповідності нормі |
| `fn_BrigadeWorkersWithEquipment` | Працівники бригади разом із закріпленою технікою |
| `fn_PlotsWithInappropriateFertilizers` | Ділянки де перевищено норму або відсутнє потрібне зрошення |
